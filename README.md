# modulor-cli
Command line tool for create necessary file for new components.


# Usage

`createComponent -n demo -p "d-" -d "./test"`

# Options

* _-n|--name_: Component Name
* _-d|--dir_: Target Directory
* _-p|--prefix_: Prefix for Component
